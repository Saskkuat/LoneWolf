Um compartimento secreto desliza para abrir na base do cajado do altar esquerdo. Dentro, você descobre uma pequena estatueta, uma Efígie de Pedra de uma estranha criatura com tentáculos.

Se você deseja manter esta Efígie, coloque-a no seu bolso e marque-a no seu Quadro de Ações como um Item Especial.

Se você deseja ativar os botões novamente, vá para 65.

Se você deseja deixar o templo, vá para 306.
