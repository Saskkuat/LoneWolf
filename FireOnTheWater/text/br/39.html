Ao anoitecer, o ônibus para em uma pousada na estrada costeira para Port Bax. O custo de um quarto para a noite é de 1 Gold Crown para passageiros do ônibus e 3 Gold Crowns para qualquer outra pessoa. Quando você está prestes a entrar, o motorista do ônibus exige ver sua passagem.

Se você tem uma passagem para a viagem para Port Bax, vire para 346.

Se você não tem uma passagem, vire para 156.
