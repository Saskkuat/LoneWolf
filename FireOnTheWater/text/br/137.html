Você chega a um cruzamento onde a Beggar Lane vira para o sul na Black Knight Street. Alguns metros adiante há outra faixa chamada Anchor Street, que segue para o leste. A chuva ficou muito mais forte agora.

Se você deseja seguir para o sul pela Black Knight Street, vire para 259.

Se você deseja seguir para o leste pela Anchor Street, vire para 20.
